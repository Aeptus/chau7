What is Chau7

Chau7 is an AI-native terminal for macOS. It detects AI agents running in the terminal, optimizes the tokens they consume, tracks their API costs, and gives them programmatic terminal access through 20 local MCP tools.

Chau7 is free, open source, and runs entirely on your Mac. It makes zero network requests to any Chau7 server. All telemetry data stays local.

The name is a French wordplay: chau + 7 (sept) = chaussette = sock.

Core Concepts

AI Agent Detection

Chau7 monitors process names in each terminal tab and matches against known AI CLI signatures. It identifies Claude Code, Codex, Gemini CLI, ChatGPT, Copilot, Aider, and Cursor automatically.

Detected agents get branded tab colors and logos. Status indicators show whether the agent is active, idle, or completed. Custom detection rules can be defined for additional CLIs.

Context Token Optimization

Context Token Optimization (CTO) strips terminal noise before AI agents read terminal output. It removes ANSI escape sequences, progress bars, spinner frames, and repeated whitespace.

CTO reduces token consumption by approximately 40% per session. It requires zero configuration and can be toggled per-tab or globally.

MCP Tools

Chau7 ships a local MCP (Model Context Protocol) server that listens on a Unix socket. AI agents connect via the standard MCP protocol and call tools to control the terminal programmatically.

20 tools cover tab management (create, list, close, execute, output), telemetry (runs, sessions, tool calls, transcripts), and resources (tab list, session list, config).

The MCP server auto-registers in Claude Code, Cursor, Windsurf, and Codex config files on every launch. No manual setup required.

API Cost Tracking

Chau7 tracks tokens, latency, and dollars per API call through a local TLS/WSS proxy. It intercepts LLM API traffic on your Mac to count tokens and estimate costs.

Cost data is shown per call, per session, and per month. All data stays local on your filesystem.

GPU-Accelerated Rendering

Chau7 renders text using Apple Metal for GPU-accelerated glyph compositing. The parsing backend uses Rust with SIMD-accelerated ANSI escape sequence processing (16-32 byte width).

A lock-free single-producer single-consumer ring buffer connects the PTY reader to the parser. Terminal state swaps atomically via triple buffering. No torn frames.

iOS Remote Control

Chau7 Remote is an iOS app that controls the Chau7 macOS terminal from an iPhone. It connects through an end-to-end encrypted relay using Curve25519 key agreement and ChaChaPoly1305 AEAD.

The relay runs on Cloudflare Workers and forwards opaque ciphertext it cannot decrypt. Features include live terminal output, command input, tab switching, approval gates, and Live Activities on the Lock Screen.

Dangerous Command Guard

Chau7 inspects command text at the terminal input layer before the shell executes it. It matches against destructive patterns (rm -rf, dd, mkfs), detects hidden Unicode characters, and catches suspicious multi-line pastes.

Flagged commands pause for user confirmation. The analysis is shell-agnostic: it works regardless of aliases, prompt themes, or shell-specific behavior.

Session Recording

Chau7 records terminal sessions with millisecond timestamps for every output event. Recorded sessions can be replayed with a visual timeline scrubber that shows terminal state at any point in time.

Session recording feature →

Privacy and Local-First Architecture

Chau7 runs entirely on your Mac. It makes zero network requests to any Chau7 server. There is no telemetry sent externally, no analytics callbacks, no crash reporting, no update checks.

Token counts, cost data, session recordings, and AI telemetry are all stored as local files on your filesystem. There is no account to create because there is no server to talk to.

MCP Tool Reference

All 20 MCP tools available over the local Unix socket.

tab_list
List all tabs across windows with status, git branch, and CTO state.
tab_create
Open a new tab with optional directory and window targeting.
tab_exec
Execute a command in a tab with prompt-state checking.
tab_status
Get process tree, telemetry run, git branch, and CTO state for a tab.
tab_send_input
Send raw input to a tab for interactive prompts (no newline appended).
tab_close
Close a tab with optional force flag for running processes.
tab_output
Get the last N lines of scrollback from a tab (max 5000 lines, 512KB cap).
tab_set_cto
Override Context Token Optimization setting for a specific tab.
run_get
Get a full telemetry run by UUID.
run_list
List runs filtered by session, repo, provider, date range, or tags.
run_tool_calls
Get every tool call an AI agent made during a run.
run_transcript
Get complete conversation turns for a run (for replay).
run_tag
Tag a run for organization and filtering.
run_latest_for_repo
Get the most recent run for a given repository path.
session_list
List sessions with run counts, filterable by repo.
session_current
Get currently active AI sessions across all tabs.
tab://list
Read-only resource: all open tabs with metadata.
session://list
Read-only resource: active AI sessions.
run://latest
Read-only resource: most recent telemetry run.
config://mcp
Read-only resource: current MCP server configuration.

Technical Architecture

Frontend
Swift + AppKit. Native macOS windowing, tab management, menus, and settings.
Parsing
Rust with SIMD-accelerated ANSI escape parsing (16-32 byte width). Lock-free SPSC ring buffer between PTY reader and parser.
Rendering
Apple Metal GPU. Glyph atlas, dirty-region redraws, triple buffering with atomic swap.
MCP Server
Local Unix socket. JSON-RPC over stdio. Auto-registers in Claude Code, Cursor, Windsurf, Codex.
iOS Remote
Cloudflare Workers relay. Curve25519 + ChaChaPoly1305 E2E encryption. Live Activities.
Privacy
100% local. Zero network requests to Chau7 servers. No telemetry, no analytics, no accounts.
