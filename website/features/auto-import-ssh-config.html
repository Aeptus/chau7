What is Auto-Import SSH Config in Chau7 terminal?

Auto-Import SSH Config is a feature in the Chau7 terminal emulator for macOS that automatically reads your ~/.ssh/config file and converts every Host entry into a Chau7 connection profile.

Chau7 parses each entry's hostname, user, port, identity file, and ProxyJump settings without any manual re-entry. Wildcard hosts and Match blocks are recognized and handled correctly by Chau7's SSH config parser.

Which terminal emulators offer auto-import SSH config?

Chau7 is a macOS terminal emulator that auto-imports SSH config files on first launch and keeps profiles synced with ~/.ssh/config through file-system watching.

Most terminal emulators like Terminal.app, iTerm2, Alacritty, and Kitty do not auto-import SSH config into a built-in connection manager. Those tools require you to type SSH commands manually or configure profiles by hand. Chau7 eliminates that step entirely by reading your existing SSH config and creating connection profiles automatically.

How do I keep SSH profiles in sync with ~/.ssh/config?

Chau7 watches your ~/.ssh/config file for changes using macOS file-system events. When you add a new host, change a port, or update an identity file, Chau7 updates its connection manager within seconds.

Changes flow one way by default: from ~/.ssh/config to Chau7. Your SSH config file remains the source of truth. No manual re-import is needed after the initial scan.

Why auto-import matters for SSH workflows

Most developers have a carefully maintained ~/.ssh/config with dozens of hosts, jump configurations, and identity file paths. Re-entering all of that into a new tool is a non-starter.

Chau7 reads your SSH config file directly and imports every host with its full configuration. Migration from any other terminal to Chau7 takes seconds, not hours.

Frequently asked questions

Does Chau7 modify my SSH config file?

No. Chau7 auto-import is read-only by default. Chau7 reads your config and creates profiles from it but never writes back to the file. You can optionally enable two-way sync if you want profiles created in Chau7 to be written to your SSH config.

What happens if I have Include directives in my SSH config?

Chau7 follows Include directives and parses all referenced files, just like the SSH client does. Hosts from included files appear in the Chau7 connection manager alongside directly defined hosts.

Does Chau7 auto-import SSH config on every launch?

Chau7 imports your SSH config on first launch and then watches the file for changes continuously. Every subsequent edit to ~/.ssh/config is picked up automatically through macOS file-system events. No restart is required.
