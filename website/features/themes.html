How it works

Chau7 ships with a curated collection of built-in themes covering popular palettes like Catppuccin, Dracula, Gruvbox, Nord, Solarized, Tokyo Night, and more. Each theme defines the full 16 ANSI colors, background, foreground, cursor, and selection colors. Switching themes applies instantly with a live preview so you can browse without committing.

Full 256-color and 24-bit true color rendering means themes look exactly as designed. Applications that emit RGB escape sequences get accurate color reproduction rather than being quantized down to the nearest ANSI color. The theme editor lets you modify any color in the palette with a native macOS color picker and see changes in real time.

iTerm2 color scheme import is supported directly. Drop an .itermcolors file onto the preferences window or use the import button, and Chau7 converts it to a native theme automatically. This gives you access to the entire iTerm2 color scheme ecosystem without manual hex value transcription.

Why it matters

A well-chosen color scheme reduces eye strain and makes syntax-highlighted output easier to scan. Chau7 removes the fragmentation of theme discovery by bundling popular schemes and supporting direct iTerm2 import, so you spend time coding instead of copying hex values between blog posts and config files.

Frequently asked questions

Can I import my existing iTerm2 color scheme?

Yes. Chau7 reads .itermcolors files natively. Import via the preferences window and the color scheme converts automatically.

Does Chau7 support true color (24-bit)?

Yes. Chau7 renders full 24-bit RGB color, so applications like delta, bat, and modern Vim colorschemes display with accurate color reproduction.

Can I create my own theme from scratch?

Yes. The built-in theme editor provides a color picker for every slot in the palette including ANSI colors, background, foreground, cursor, and selection.
