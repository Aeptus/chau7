What is Themes in Chau7?

Themes in Chau7 is a built-in color scheme system that ships with a curated collection of popular palettes. Chau7 includes Catppuccin, Dracula, Gruvbox, Nord, Solarized, Tokyo Night, and more.

Each Chau7 theme defines the full 16 ANSI colors, background, foreground, cursor, and selection colors. Switching themes in Chau7 applies instantly with a live preview so developers can browse without committing to a choice.

How to import an iTerm2 color scheme into Chau7

Chau7 reads .itermcolors files natively. Drop an iTerm2 color scheme file onto the Chau7 preferences window or use the import button.

Chau7 converts iTerm2 color schemes to native Chau7 themes automatically. No manual hex value transcription is needed. This gives Chau7 users access to the entire iTerm2 color scheme library.

How to customize terminal colors beyond 16 ANSI in Chau7

Chau7 supports full 256-color and 24-bit true color rendering. Applications that emit RGB escape sequences get accurate color reproduction in Chau7 rather than being quantized down to the nearest ANSI color.

Chau7's built-in theme editor provides a native macOS color picker for every slot in the palette. Developers can modify all 16 ANSI colors, background, foreground, cursor, and selection colors. Changes preview in real time so the result is visible before saving.

Does Chau7 support true color (24-bit)?

Yes. Chau7 renders full 24-bit RGB color. Applications like delta, bat, and modern Vim colorschemes display with accurate color reproduction in Chau7.

Many older terminals only support 256 colors, which forces applications to approximate colors. Chau7's true color support means themes look exactly as designed, with no color quantization artifacts.

Can I create my own Chau7 theme from scratch?

Yes. Chau7's built-in theme editor provides a color picker for every slot in the palette, including all ANSI colors, background, foreground, cursor, and selection.

The Chau7 theme editor shows changes in real time. Developers can experiment with colors and see the result immediately in the terminal, without editing configuration files or restarting.
