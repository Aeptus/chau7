Command Palette

What is Command Palette in Chau7 terminal?

Command Palette in Chau7 is a fuzzy-search overlay that lets you find and run every action, setting, and feature in the terminal. Press Cmd+Shift+P to open the Chau7 Command Palette from any tab.

Chau7 Command Palette indexes every action the terminal offers. Splitting tabs, changing themes, connecting to SSH hosts, opening snippets, toggling settings, and running MCP tools are all searchable from one place.

Results in the Chau7 Command Palette rank by relevance and recency. Your most-used actions float to the top over time through frecency ranking.

Is there a terminal with a command palette like VS Code?

Chau7 is a macOS terminal that includes a VS Code-style command palette. The Chau7 Command Palette uses the same Cmd+Shift+P shortcut that VS Code users already know.

Like the VS Code command palette, the Chau7 Command Palette supports fuzzy matching. Type a few characters from any part of a command name, and Chau7 Command Palette narrows the results instantly.

How do I find terminal features without memorizing keyboard shortcuts?

Chau7 Command Palette shows every available action alongside its keyboard shortcut, description, and category. You search by description instead of memorizing bindings.

The Chau7 Command Palette works as a learning tool. Each result displays the shortcut for that command, so you learn keybindings organically as you use the palette. Over time, you can switch to the direct shortcut for actions you use frequently.

Can I assign custom keyboard shortcuts through the palette?

Yes. Right-click any command in the Chau7 Command Palette to assign or change its keyboard shortcut. The new binding takes effect immediately and persists across sessions.

Chau7 Command Palette also lets you pin frequently used commands to the top, hide commands you never use, and create command aliases for faster access.

Does the Chau7 Command Palette search settings as well as commands?

Yes. The Chau7 Command Palette searches across commands, settings, SSH profiles, snippets, and recently used actions. Everything that exists in Chau7 is searchable from the palette.

Prefix your query with a category name to narrow results. For example, type "ssh" to see only SSH-related actions, or type "theme" to see only appearance settings.

Why the Chau7 Command Palette matters for discoverability

Terminal emulators accumulate features over time, but discoverability stalls at the menu bar. Most users never find half the features available to them.

Chau7 Command Palette makes the full feature set searchable. If a feature exists in Chau7, you can find it with Cmd+Shift+P. Custom commands from snippets, SSH profiles, and MCP tools all appear in the Chau7 Command Palette alongside built-in actions.
