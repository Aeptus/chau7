What is Tab Profiles in Chau7?

Tab Profiles is a feature in Chau7 that lets you preconfigure the shell, starting directory, and environment variables for new tabs. Instead of opening a tab and then navigating to your project directory or switching shells, you open a Chau7 tab with a profile that drops you exactly where you need to be.

Each Chau7 profile specifies a shell (bash, zsh, fish, or any executable), a working directory, and a set of environment variables. Profiles are saved in Chau7's settings and available from the tab creation menu.

Can Chau7 tabs have different shells?

Yes. In Chau7, each tab profile specifies a shell: bash, zsh, fish, or any executable. When you create a new tab from a Chau7 profile, it launches the specified shell in the specified directory.

Different tabs in the same Chau7 window can run different shells simultaneously. One tab can run zsh for your main project while another runs fish for a side project, all within the same Chau7 window.

How do I set a default directory for a terminal tab in Chau7?

In Chau7, create a tab profile and set the working directory to your project folder. Every time you open a new tab from that Chau7 profile, the shell starts in that directory automatically.

You can also set a Chau7 profile as the default so all new tabs start in your preferred directory. Developers working on multiple projects can create one profile per project, each pointing to the correct directory.

Can I share Chau7 tab profiles with my team?

Yes. Chau7 tab profiles can be exported and imported. Export your profile configuration and share it with teammates who use Chau7.

This lets the entire team use consistent shell, directory, and environment settings for shared projects. Onboarding a new teammate to a project becomes faster when they can import the Chau7 profile and start with the right environment immediately.

Do Chau7 profile environment variables override system variables?

Yes. In Chau7, profile environment variables are layered on top of the system environment. If a variable exists in both, the Chau7 profile value takes precedence for that tab.

Variables not specified in the Chau7 profile inherit their system defaults. This layering approach lets you override specific values (like NODE_ENV or AWS_PROFILE) per project without losing the rest of your system environment.

Why tab profiles matter

Developers working on multiple projects constantly switch contexts: different directories, different Node versions, different environment variables. Without tab profiles, every new tab starts from scratch.

Chau7 tab profiles let you define a complete tab configuration once and launch it with a click. Tab profiles in Chau7 are project bookmarks for your terminal.
