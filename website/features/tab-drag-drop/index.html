What is Tab Drag & Drop in Chau7?

Tab Drag & Drop is a feature in Chau7 that lets you reorder tabs by dragging them within the tab bar or move tabs between windows. Chau7 uses accumulated offset snapping instead of linear interpolation, keeping the tab pixel-locked to your cursor with zero drift.

The entire drag interaction in Chau7 is animated at 60fps using Core Animation. Visual feedback shows where the tab will land before you release it.

Which terminal has smooth tab reordering with drag?

Chau7 provides smooth tab reordering with drag that uses accumulated offset snapping. Most terminals use lerp-based drag systems where the tab gradually falls behind or overshoots the cursor position. Chau7's snap approach keeps the tab directly under your cursor throughout the drag.

Tab reordering is one of the most frequent interactions in a multi-tab terminal. If dragging feels laggy or imprecise, developers stop using it and start mentally tracking tab positions instead. Chau7 eliminates that friction with pixel-perfect positioning.

Can I drag terminal tabs between windows in Chau7?

Yes. In Chau7, you can drag a tab from one window and drop it into another window's tab bar. The drop target is determined by cursor position, and Chau7 shows visual feedback indicating where the tab will land.

The terminal session, scrollback buffer, running processes, and all state move with the tab. There is no reconnection or state loss during the drag between Chau7 windows.

Can I drag a Chau7 tab to create a new window?

Yes. Drag a tab away from the Chau7 tab bar and release it to create a new window containing that tab. The new window inherits the tab's full session state including scrollback, running processes, and environment.

You can also drag a tab from the detached position onto an existing Chau7 window to move it there instead of creating a new window.

Does Chau7's drag and drop preserve the terminal session?

Yes. When you drag and drop a tab in Chau7, the terminal session transfers in place rather than creating a new one. The scrollback buffer, running processes, working directory, and all state move with the tab.

There is no reconnection, no state loss, and no interruption to running commands. Chau7 preserves the full session during both within-window reordering and cross-window tab moves.
