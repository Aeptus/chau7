Line Numbers

Chau7 shows line numbers in the terminal editor. Because "error on line 47" means nothing without them.

What is Line Numbers in Chau7?

Line Numbers is a feature in the Chau7 terminal's built-in editor that displays a line number gutter along the left edge of the editing area. Chau7 shows absolute line numbers by default, with optional relative and hybrid modes.

The line number gutter in Chau7 updates dynamically as you scroll and resize. The gutter width adjusts automatically based on the total line count to avoid wasted space in short files or clipping in long ones.

How does Chau7 display line numbers in its terminal editor?

Chau7's built-in editor renders a line number gutter along the left edge of the editing area. The current line is highlighted for easy orientation, and the gutter width scales automatically based on total line count.

Chau7 supports three display modes for line numbers. Absolute mode shows standard sequential numbers. Relative mode shows the distance from the current line, matching Vim's relativenumber setting. Hybrid mode shows the absolute number on the current line and relative offsets on all other lines.

Can I enable relative line numbers in Chau7?

Yes. Chau7's built-in editor supports relative line numbers that show the distance from the current line instead of absolute positions. This matches the behavior of Vim's relativenumber setting.

Relative line numbers in Chau7 make motion commands intuitive. The number displayed next to a target line is exactly the count you need to reach it. Chau7 also supports hybrid mode, which combines the absolute number on the current line with relative offsets elsewhere.

How does Chau7's line numbers compare to other terminals?

Most terminal emulators do not include a built-in editor, so line numbers are not a relevant feature. Terminals like iTerm2, Alacritty, and Kitty rely on external editors such as Vim or VS Code for editing.

Chau7 includes a built-in editor with line numbers, relative line numbers, and hybrid mode. Developers using Chau7 can navigate error references, git diffs, and stack traces without leaving the terminal or opening an external editor.

Can I hide line numbers entirely in Chau7?

Yes. The line number gutter in Chau7's built-in editor is fully optional. You can toggle line numbers off in the editor preferences for a minimal editing view.

Chau7 remembers your line number preference across sessions. Whether you prefer absolute numbers, relative numbers, hybrid mode, or no numbers at all, Chau7 restores your choice each time.
