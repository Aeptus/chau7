How it works

Chau7 ships with over 100 monospace fonts embedded directly in the application bundle, including popular choices like JetBrains Mono, Fira Code, Cascadia Code, Iosevk, and Hack. Nerd Font variants are included out of the box, so powerline symbols, file icons, and dev glyphs render correctly without downloading or installing anything. The font picker shows a live preview of each font at your current size and weight settings.

Variable font support means you can dial in exactly the weight you want rather than picking from a handful of static weights. Adjust from hairline to black in continuous increments using a slider. Font size scales smoothly with pinch-to-zoom or keyboard shortcuts, and per-profile font configuration lets you use different typefaces for different workflows.

System fonts are fully supported alongside bundled fonts. If you have a custom typeface installed at the OS level, Chau7 detects and offers it in the font picker. Font rendering uses Core Text natively for sharp subpixel antialiasing on Retina and non-Retina displays alike.

Why it matters

Font choice directly affects readability during long coding sessions. Chau7 eliminates the friction of finding, downloading, and installing terminal fonts by bundling everything developers actually use. Variable weight support and per-profile configuration let you fine-tune legibility for different monitors, lighting conditions, and personal preference without leaving the app.

Frequently asked questions

Can I use my own custom fonts?

Yes. Chau7 detects any monospace font installed on your system and makes it available in the font picker alongside the 100+ bundled fonts.

Do Nerd Font glyphs work out of the box?

Yes. Chau7 bundles Nerd Font variants so powerline symbols, devicons, and other special glyphs render without any manual font installation.

Can I set different fonts per profile?

Yes. Each profile stores its own font family, size, and weight, so you can use different typography for SSH sessions, local development, or presentations.
