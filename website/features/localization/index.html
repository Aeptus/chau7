Localization (4 Languages)

Your terminal, in your language. Because not everyone thinks in English.

Questions this answers

  • Terminal app localized languages macOS
  • RTL right-to-left terminal emulator support
  • Change terminal app language without restart
  • Non-English terminal emulator UI macOS

How it works

Chau7 is fully localized in four languages, with every menu item, preference label, dialog, error message, and accessibility description translated. The application follows the macOS system language by default, so it appears in your preferred language automatically. If your system language is set to a supported language, Chau7's interface matches without any configuration.

Runtime language switching lets you change the interface language without restarting the application. This is useful for developers who work in multilingual environments or prefer their tools in a different language from their system setting. All UI elements update immediately including menus, tab labels, and the preferences window.

Right-to-left language support covers the complete UI layout. Menus, tab bars, preferences panels, and text alignment all mirror correctly for RTL languages. The terminal content area itself remains left-to-right since terminal protocols are LTR, but all application chrome respects the RTL writing direction.

Why it matters

Developers work in every language, and a terminal that only speaks English creates unnecessary friction. Chau7 supports localization for its UI, menus, settings, and documentation. The terminal speaks your language so you can focus on your code.

Frequently asked questions

Which languages are supported?

Chau7 is currently localized in four languages. The application follows your macOS system language automatically and supports runtime switching between all available languages.

Does RTL support affect terminal content?

Terminal content remains left-to-right since terminal protocols are inherently LTR. RTL support applies to all application chrome: menus, tabs, preferences, dialogs, and accessibility labels.

Can I switch languages without restarting?

Yes. Language switching is instant at runtime. All menus, labels, and UI elements update immediately without requiring an application restart.
