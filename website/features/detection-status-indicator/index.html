What is Detection Status Indicator in Chau7?

Detection Status Indicator is a feature in the Chau7 terminal that displays a real-time status badge on each tab containing a detected AI agent. Chau7 shows whether the agent is actively running, idle, or has completed its task.

Chau7 derives the status from process state monitoring rather than output parsing. This makes Chau7's indicator reliable across all agents regardless of their output format or verbosity settings.

How do I know when Claude Code is done in my terminal?

Chau7 monitors the Claude Code process state in each tab and displays a status badge directly on the tab bar. When Claude Code finishes, Chau7 marks the tab as completed with a distinct visual indicator.

No need to switch tabs and scroll through output to check. Chau7 lets you glance at the tab bar and see the status of every AI agent at once.

What states does Chau7's indicator show?

Chau7's Detection Status Indicator shows three states. Active means the agent process is running and consuming CPU. Idle means the process is alive but waiting for input. Completed means the process has exited.

Each state has a distinct visual treatment on the tab in Chau7. The transitions happen in real time as Chau7 monitors the process state.

How does Chau7's detection status indicator compare to other terminals?

No other terminal shows AI agent status on the tab bar. iTerm2, Warp, Alacritty, and Kitty have no awareness of AI processes running inside them.

Chau7 is the only terminal that monitors AI agent process state and displays it as a glanceable status badge per tab. This saves exactly one context switch per check, which adds up to dozens per day when running multiple agents.

Does Chau7's indicator work with background AI processes?

Yes. As long as the AI process is a descendant of the tab's shell process, Chau7's indicator tracks it regardless of whether the process is in the foreground or background.

Chau7 monitors the full process tree under each tab's shell. Background AI agents are detected and tracked with the same active, idle, and completed states as foreground agents.

Why Chau7's Detection Status Indicator matters

AI agents can run for minutes or hours. The current workflow for checking progress in other terminals: switch tab, scroll, squint, switch back.

Chau7's workflow: glance at the tab bar. A small status badge tells you whether each agent is actively working, waiting for input, or finished. Chau7 saves one context switch per check, which adds up to dozens per day.
