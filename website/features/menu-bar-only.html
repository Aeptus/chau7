Menu Bar Only

Chau7 hides from the Dock and lives in the menu bar. One hotkey away. Always available, never in the way.

What is Menu Bar Only in Chau7?

Menu Bar Only is a window mode in Chau7 that removes the app from the macOS Dock entirely and places a small icon in the menu bar instead. Clicking the Chau7 menu bar icon or pressing a global hotkey reveals the terminal window.

When Chau7 is in menu bar only mode, the application stays running in the background with zero visual footprint beyond the menu bar icon. Chau7 does not appear in Cmd+Tab or the Dock. All terminal sessions persist while the window is hidden.

How to run Chau7 as a menu bar app with no Dock icon

Enable menu bar only mode in Chau7's preferences. Chau7 immediately disappears from the Dock and appears as a menu bar icon. No restart is required.

The Chau7 menu bar icon shows a subtle status indicator when commands are running or when terminal bell notifications fire. Right-clicking the Chau7 menu bar icon provides quick access to recent sessions, new tab creation, and preferences.

Which terminal emulators offer menu bar only mode?

Chau7 offers a native menu bar only mode on macOS. iTerm2 has a similar option. Alacritty, Kitty, and Warp do not support running as a menu bar only app.

Chau7's menu bar mode is unique because it combines with Chau7's Quake-style dropdown and overlay mode. Developers can run Chau7 in the menu bar and summon the terminal with a global hotkey that triggers a dropdown from the top of the screen.

Does Chau7 still run processes when hidden in the menu bar?

Yes. All terminal sessions in Chau7 persist in the background when the window is hidden. Running processes, SSH connections, and shell state are maintained whether Chau7's window is visible or not.

Chau7's menu bar only mode does not affect terminal functionality. Long-running builds, SSH tunnels, and background tasks continue without interruption. Chau7 only hides the window, not the terminal sessions.

Can I combine Chau7's menu bar mode with the dropdown terminal?

Yes. This is a popular combination in Chau7: Chau7 lives in the menu bar with no Dock presence, and a global hotkey triggers the Quake-style dropdown when you need it. Chau7 stays invisible until summoned.

To set up this combination in Chau7, enable both menu bar only mode and dropdown mode in preferences, then assign a global hotkey. Chau7 will have no Dock icon, no Cmd+Tab entry, and a one-keystroke dropdown terminal available from any application.
