Menu Bar Only

No Dock icon. One hotkey away. Always available, never in the way.

Questions this answers

  • Run terminal as menu bar app macOS no Dock icon
  • Hide terminal from Dock macOS
  • Menu bar terminal access hotkey macOS
  • Background terminal app macOS always running

How it works

Menu bar only mode removes Chau7 from the Dock entirely and places a small icon in the macOS menu bar. Clicking the menu bar icon or pressing a global hotkey reveals the terminal window. When the terminal loses focus or you press the hotkey again, it hides. The application stays running in the background with zero visual footprint beyond the menu bar icon.

This mode pairs naturally with dropdown or overlay mode. Configure Chau7 to run in the menu bar with a Quake-style dropdown, and you get an always-available terminal that never clutters your Dock or appears in Cmd+Tab. All terminal sessions persist in the background, so processes keep running whether the window is visible or not.

The menu bar icon shows a subtle status indicator when commands are running or when terminal bell notifications fire, so you maintain awareness without the window being visible. Right-clicking the menu bar icon provides quick access to recent sessions, new tab creation, and preferences.

Why it matters

Developers who use the terminal intermittently don't need it occupying a permanent Dock slot. Menu bar only mode keeps Chau7 accessible via a global hotkey or menu bar icon without cluttering the Dock or the Cmd+Tab app switcher. The terminal is one keystroke away but invisible when you don't need it.

Frequently asked questions

Does Chau7 still run processes when hidden in the menu bar?

Yes. All terminal sessions persist in the background. Running processes, SSH connections, and shell state are maintained whether the window is visible or not.

Can I combine menu bar mode with the dropdown terminal?

Yes. This is a popular combination: Chau7 lives in the menu bar with no Dock presence, and a global hotkey triggers the Quake-style dropdown when you need it.

How do I switch back to normal Dock mode?

Toggle the menu bar only setting in preferences. Chau7 immediately reappears in the Dock without restarting the application.

Hide your terminal from the Dock. It'll still be there when you need it.
