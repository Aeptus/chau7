Timeline Scrubber

Chau7 lets you scrub through terminal recordings like a video. Jump to any moment. See exactly what happened.

The problem

  • Huge recordings are painful to scrub through using raw scroll alone.
  • Developers need a fast way to jump to the moment a command or failure happened.

What Chau7 does about it

  • Displays a visual horizontal timeline for recorded sessions.
  • Marks command segments so navigation is tied to meaningful activity, not just line count.
  • Supports scrub-style replay and quick jumping across the session.
  • Shares analytics and session-recording data rather than duplicating its own store.

What is Timeline Scrubber in Chau7?

Timeline Scrubber is a visual navigation tool in Chau7 for recorded terminal sessions. Chau7's scrubber displays command blocks as colored segments on a timeline, making it easy to see where different commands were executed.

The timeline minimap now includes an activity density overlay that buckets command blocks into 4px columns and varies the accent color opacity by event count, showing activity hotspots at a glance.

Developers can click anywhere on Chau7's timeline to jump to that moment, or drag the playhead to scrub through the session and watch the terminal state change in real time. At any point, Chau7 reconstructs the exact terminal state from the recording data.

How do I replay a terminal session at a specific point in time?

In Chau7, open a recorded session and click anywhere on the timeline scrubber to jump to that moment. Chau7 reconstructs the exact terminal state at that timestamp, including every character, color, and cursor position.

Chau7's playback supports variable speed: 0.5x for slow inspection, 1x for realistic replay, 2x and 4x to skip idle sections. Developers can play, pause, and examine the full terminal state at any point.

How does Chau7's timeline scrubber compare to other terminals?

Most terminals do not record sessions at all. Tools like asciinema offer linear playback but no visual timeline navigation. Chau7's timeline scrubber provides click-to-seek, drag-to-scrub, variable-speed playback, and full terminal state reconstruction at any point.

No other terminal offers this level of session navigation. Chau7's timeline scrubber makes recorded sessions as easy to navigate as a video player, replacing the linear text-dump approach used by other recording tools.

Can I search within a recorded session in Chau7?

Chau7's timeline scrubber provides visual navigation through recorded sessions. The timeline shows command blocks as colored segments that help developers quickly identify different command executions.

Developers can click on any command block segment to jump to that moment and inspect the terminal state. This lets developers find specific commands without scanning through the entire recording linearly.

Does the Chau7 timeline show command boundaries?

Yes. Chau7's timeline scrubber shows command blocks as colored segments on the timeline, making it easy to identify individual command executions visually.

Developers can navigate between command segments in Chau7's timeline by clicking on them directly. Each segment corresponds to a command and its output, providing clear visual boundaries.
