Command Palette

Chau7 Command Palette puts ~60 commands one search away. Press Cmd+Shift+P to open it.

The problem

  • Mouse-hunting for commands and settings wastes the benefit of a keyboard-first terminal.
  • Power features are useless if users cannot discover them quickly.

What Chau7 does about it

  • Fuzzy-searchable command palette similar to editor launchers.
  • Keyboard-first action execution keeps common operations one shortcut away.
  • Search state, clearing, and selection are handled inside the palette itself.
  • Palette actions can trigger operational commands like snippet reloads and other advanced flows.

What is Command Palette in Chau7 terminal?

Command Palette in Chau7 is a search overlay that lets you find and run commands in the terminal using substring matching. Press Cmd+Shift+P to open the Chau7 Command Palette from any tab.

Chau7 Command Palette includes approximately 60 commands organized into categories. Each command displays its keyboard shortcut when available, making the palette useful for both executing actions and discovering keybindings.

Commands are statically ordered in the Chau7 Command Palette. Type to filter results by substring matching.

Is there a terminal with a command palette like VS Code?

Chau7 is a macOS terminal that includes a VS Code-style command palette. The Chau7 Command Palette uses the same Cmd+Shift+P shortcut that VS Code users already know.

Like the VS Code command palette, the Chau7 Command Palette lets you type to filter commands by name. Chau7 uses substring matching to narrow the results as you type.

How do I find terminal features without memorizing keyboard shortcuts?

Chau7 Command Palette shows every available action alongside its keyboard shortcut, description, and category. You search by description instead of memorizing bindings.

The Chau7 Command Palette works as a learning tool. Each result displays the shortcut for that command, so you learn keybindings organically as you use the palette. Over time, you can switch to the direct shortcut for actions you use frequently.

How many commands does the Chau7 Command Palette include?

The Chau7 Command Palette includes approximately 60 commands organized into categories. Commands cover tab management, appearance, session operations, and other built-in actions.

Each command displays its keyboard shortcut when available, so developers can learn keybindings by browsing the palette.

Why the Chau7 Command Palette matters for discoverability

Terminal emulators accumulate features over time, but discoverability stalls at the menu bar. Most users never find half the features available to them.

Chau7 Command Palette makes the built-in command set searchable. Press Cmd+Shift+P and type to find any of the ~60 built-in commands by name.
