Nature du site

Le site chau7.sh est un site public de présentation, de comparaison et de documentation marketing autour du terminal Chau7. Il renvoie notamment vers le dépôt GitHub du projet, les notes de version, les pages fonctionnalités, les pages de performance et les informations légales associées.
